## 🛠️ 技术栈

- **前端：** HTML5, CSS3, JavaScript (ES6+)
- **样式：** Tailwind CSS, 自定义CSS变量
- **动画：** CSS Animations, Canvas粒子效果
- **后端：** Vercel Serverless Functions
- **AI服务：** 智谱AI (GLM-4-Flash)
- **部署：** Vercel
