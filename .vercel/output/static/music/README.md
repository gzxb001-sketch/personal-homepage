# 音乐文件夹

此文件夹用于存放背景音乐文件。

## 如何更换音乐

1. 将你的音乐文件（支持 MP3、WAV、OGG 等格式）放入此文件夹
2. 推荐文件大小：不超过 5MB
3. 推荐时长：循环播放的轻音乐，30秒 - 3分钟

## 使用方法

在 `spa.html` 中修改音乐路径（约第 350 行）：

```javascript
// 修改这里：
this.bgMusic.src = 'music/你的音乐文件名.mp3';
```

## 推荐音乐类型

- Lo-fi Hip Hop
- 轻音乐
- 环境音乐
- 氛围音乐
- 轻爵士
